## About
Alpaca is a tool to manage/autimate an apache server.

## Install

clone repo
`git clone https://github.com/jacobnelsonfoale/alpaca-apache`

make file executable
`chmod +x alpaca`

move to bin folder (so you can execute anywhere)
`cp alpaca /bin/`

help
`alpaca help`
